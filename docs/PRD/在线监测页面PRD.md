## 一、产品背景与目标

### 1.1 产品定位
在线监测知识库的核心模块，面向电力、新能源、工业制造、石化、轨道交通等行业的设备管理工程师、技术人员、运维人员，提供全面、系统的在线监测技术知识体系。

#### 核心问题
1. **内容深度不一**：开关柜监测内容详实，但GIS、电缆、避雷器等内容过于简略
2. **缺少振动监测专题**：振动监测是设备健康管理的重要组成部分，当前缺失
3. **可视化不足**：缺乏交互式图表、动画演示、流程图等可视化元素
4. **学习体验弱**：内容以文字为主，不适合系统性学习和知识传递
5. **知识孤立**：各监测系统之间缺乏关联，未形成知识图谱

### 2.2 核心内容规划

#### 2.2.1 新增：振动监测专题（★★★★★ 优先级）

**内容大纲**：

1. **振动监测基础**
   - 振动基本概念：位移、速度、加速度
   - 振动传感器原理：压电式、MEMS、IEPE
   - 信号处理基础：采样定理、滤波、FFT

2. **振动监测参数**
   - 加速度峰值（RMS、峰值）
   - 振动频率特征
   - 频谱分析（FFT）
   - 包络分析（ envelope）

3. **设备故障特征频率**
   ```
   滚动轴承故障：
   - 外圈故障频率 = (n/2) × (1 - d/D × cosθ) × RPM/60
   - 内圈故障频率 = (n/2) × (1 + d/D × cosθ) × RPM/60
   - 滚动体故障频率 = (D/d) × (1 - (d/D)² × cos²θ) × RPM/60

   齿轮故障：
   - 齿轮啮合频率 = 齿数 × RPM/60
   - 边带频率 = 啮合频率 ± k × 轴旋转频率
   ```

4. **工业应用场景**
   - 旋转机械（电机、压缩机、泵）
   - 变压器铁芯振动
   - 开关柜机械特性
   - 管道振动与共振

5. **振动分析案例**
   - 轴承故障诊断实例（含频谱图）
   - 不平衡、不对中故障识别
   - 共振检测与处理

## 三、页面设计规范

### 3.1 内容呈现规范

1. **信息层次**
   - 一级标题：章节号 + 主标题
   - 二级标题：重要概念
   - 三级标题：具体参数/说明
   - 正文：详细解释

2. **数据展示**
   - 使用表格：参数对比、规格表
   - 使用卡片：多个并列概念
   - 使用列表：步骤、要点
   - 使用图表：数据变化、趋势

3. **交互元素**
   - 卡片点击：展开详情
   - 图表悬停：显示数值
   - 标签点击：快速筛选
   - 目录跳转：平滑滚动

### 3.2 视觉设计建议

```
配色方案：
- 主色：#6366F1 (品牌紫)
- 辅助色：#8B5CF6 (渐变紫)
- 强调色：#10B981 (成功绿) / #F59E0B (警告黄) / #EF4444 (危险红)
- 背景色：#0A0E1A (深色) / #F8FAFC (浅色)

字体：
- 标题：PingFang SC / Noto Sans SC
- 正文：PingFang SC / Noto Sans SC
- 代码/参数：JetBrains Mono / Courier New

间距：
- 区块间距：32px
- 卡片间距：16px
- 内部间距：16px/24px
```
