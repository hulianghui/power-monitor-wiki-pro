## 0. 产品概述（V1.0-Final）

### 0.1 核心定位

> **电力在线监测知识库 = 科技感知识平台 + RFID 核心技术 + AI 智能助手**
>
> **设计理念**：科技感视觉风格 + 双板块并行结构 + 探索式学习体验

### 0.2 目标用户

| 用户角色 | 描述 | 核心需求 |
|----------|------|----------|
| **RFID 技术人员** | 从事 RFID 无源无线测温产品的技术人员 | 深入了解 RFID 技术原理、产品体系、应用场景 |
| **电力监测工程师** | 电力系统运维、监测的技术人员 | 学习在线监测技术、故障诊断方法 |
| **项目技术人员** | 需要综合了解两大技术方向的人员 | 快速获取两大技术方向的核心知识 |
| **行业新人** | 刚入行的技术人员 | 系统性了解行业技术框架 |

### 0.3 核心价值

```
1. RFID 无源无线测温技术 → 独立完整展示（核心业务）
2. 电力在线监测技术 → 独立完整展示（技术方向）
3. 科技感视觉体验 → 深色主题 + 渐变光效 + 流畅动效
4. AI 智能助手 → 随时在线解答疑问
```

## 3. 视觉设计规范

### 3.1 色彩系统

```css
/* 背景色 */
--bg-dark: #0a0e1a;           /* 深空黑 */
--bg-card: #111827;           /* 卡片背景 */
--bg-card-hover: #1f2937;     /* 卡片悬停 */

/* 品牌色 */
--brand-primary: #6366f1;     /* 科技蓝紫 */
--brand-secondary: #8b5cf6;   /* 紫罗兰 */
--brand-accent: #06b6d4;      /* 青色点缀 */
--brand-glow: rgba(99, 102, 241, 0.3);

/* RFID 专属色 */
--rfid-primary: #a855f7;      /* 霓虹紫 */
--rfid-secondary: #ec4899;    /* 粉紫 */
--rfid-glow: rgba(168, 85, 247, 0.4);

/* 监测专属色 */
--monitor-primary: #06b6d4;   /* 赛博青 */
--monitor-secondary: #3b82f6; /* 科技蓝 */
--monitor-glow: rgba(6, 182, 212, 0.4);

/* 文字 */
--text-primary: #f8fafc;      /* 主文字 */
--text-secondary: #94a3b8;    /* 次要文字 */
--text-tertiary: #64748b;     /* 辅助文字 */

/* 边框 */
--border: rgba(148, 163, 184, 0.1);
--border-glow: rgba(99, 102, 241, 0.3);
```

### 3.2 字体系统

```css
--font-family: "PingFang SC", "SF Pro SC", -apple-system, BlinkMacSystemFont, sans-serif;
```
